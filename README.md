## Installation
To install Zyvvven, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/SebastianMou/Zyvvven.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Zyvvven
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

To start using Zyvvven, run:
